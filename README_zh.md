# 贪吃蛇游戏

使用 Godot 4 引擎实现的经典贪吃蛇游戏。

## 功能特性

- 经典贪吃蛇游戏机制
- 分数追踪
- 游戏结束检测
- 响应式控制
- 简洁清晰的界面

## 项目结构

```
snake/
├── scene/
│   ├── food/          # 食物对象及其逻辑
│   ├── hud/           # 分数显示界面
│   ├── main/          # 主游戏场景及其逻辑
│   └── snake/         # 蛇对象及其逻辑
├── .editorconfig      # 编辑器配置
├── .gitattributes     # Git 属性
├── .gitignore         # Git 忽略规则
├── icon.svg           # 游戏图标
├── project.godot      # Godot 项目配置
├── README.md          # 英文说明文件
└── README_zh.md       # 中文说明文件
```

## 如何运行

1. 安装 Godot 4 或更高版本
2. 在 Godot 中打开项目
3. 按 F5 运行游戏

## 控制方式

- **方向键**：控制蛇的移动方向
- **空格键**：游戏结束后重新开始

## 开发信息

本项目使用 Godot 4 开发。代码被组织成单独的场景和脚本，以提高可维护性。
